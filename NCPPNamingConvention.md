## Prefixes: ##
+ **I_** : Interface (prefer to use as inheritable class).
+ **C_** : Non-inheritable Class
+ **m_** : Member variable (prefer to use for private or protected member only).
+ **S_** : Struct
+ **E_** : Enum
+ **g_** : Global Variable
+ **TC_** : Class template
+ **HN_** : Helper namespace
